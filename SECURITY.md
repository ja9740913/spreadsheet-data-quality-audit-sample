# Security

This repository contains only fictional spreadsheet samples, generated reports, documentation, images, checksums, and a verifier.

- It contains no macros, executable workbook content, paid audit engine, PowerShell launcher, customer data, credentials, API keys, or private operating files.
- Do not commit a real customer or employer workbook, output, screenshot, filename, email address, phone number, receipt, or account detail.
- Treat downloaded XLSX files according to your normal organizational controls. The verifier checks the container and rejects macro and external-link payloads, but it is not an antivirus product.
- Report a suspected public-data leak through GitHub's private vulnerability reporting feature if it is enabled. Do not post the suspected sensitive value in a public issue.

The paid toolkit processes authorized files locally. Its separate product documentation and versioned terms control supported environments, data handling, and license scope.
